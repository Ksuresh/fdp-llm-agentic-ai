# Reading List & Resources

A curated list to keep you current after the FDP. Organised by topic.

---

## 📖 Must-Read Papers

| Paper | Why It Matters |
|-------|---------------|
| [Attention Is All You Need](https://arxiv.org/abs/1706.03762) (Vaswani et al., 2017) | The transformer architecture that powers all modern LLMs |
| [DeepSeek-R1](https://arxiv.org/abs/2501.12948) (2025) | How DeepSeek achieved o1-level reasoning at $5.6M training cost using RLVR |
| [Constitutional AI](https://arxiv.org/abs/2212.08073) (Anthropic, 2022) | How Anthropic trains Claude using AI feedback instead of human raters |
| [ReAct: Reasoning + Acting](https://arxiv.org/abs/2210.03629) (Yao et al., 2022) | The foundation of how modern AI agents think and act |
| [LoRA](https://arxiv.org/abs/2106.09685) (Hu et al., 2021) | The efficient fine-tuning technique we use in Day 2 |
| [RAGAS](https://arxiv.org/abs/2309.15217) (Es et al., 2023) | RAG evaluation framework used in Day 2 |
